# Simulacro de Phishing - Seguridad Coautonoma CTA

Este repositorio contiene la página de destino para las campañas de simulación de phishing y concientización sobre ciberseguridad de **Coautonoma CTA**.

## 🎯 Objetivo del Proyecto

El propósito de esta página es educar de manera interactiva a los colaboradores de la organización. Cuando un usuario hace clic en un enlace de prueba y proporciona sus datos, el sistema registra el evento y le muestra una advertencia educativa detallada sobre los riesgos del phishing.

## 🚀 Características

- **Diseño Corporativo Confiable:** Interfaz moderna y limpia diseñada con fuentes institucionales (Outfit) y colores de la marca para simular un portal real de validación de red interna.
- **Formulario de Validación:** Captura de forma segura el usuario corporativo y correo institucional del colaborador.
- **Registro de Métricas:** Integración con Google Forms para registrar de forma anónima o controlada los ingresos durante el simulacro.
- **Retroalimentación Inmediata:** Muestra una advertencia de seguridad con consejos de prevención una vez que se envían los datos.
- **Google Analytics:** Configurado para medir el tráfico y efectividad de la campaña de concientización.

## 🛠️ Tecnologías Utilizadas

- **HTML5 & CSS3:** Maquetación responsiva con variables personalizadas y animaciones fluidas.
- **JavaScript (Vanilla):** Gestión del envío del formulario (vía `fetch` / `no-cors`) y transición de secciones dinámicas.

---
*Nota: Este proyecto tiene fines estrictamente educativos y de concientización de seguridad de la información.*
